## 🎨 页面设计规范

### 整体设计风格
- **设计理念**：简洁现代、专业高效、用户友好
- **主色调**：深蓝色 (#1890FF) + 白色 (#FFFFFF)
- **辅助色**：灰色系 (#F5F5F5, #E8E8E8, #999999)
- **强调色**：成功绿 (#52C41A)、警告橙 (#FA8C16)、错误红 (#F5222D)
- **字体**：系统默认字体栈，主要使用 -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- **圆角**：统一使用 6px 圆角
- **阴影**：轻微阴影效果，提升层次感

### 关键页面设计

#### 1. 🏠 仪表板首页
**布局结构**：
- 顶部导航栏（固定高度 60px）
- 左侧菜单栏（宽度 240px，可折叠）
- 主内容区域（自适应宽度）

**主要元素**：
- **数据概览卡片**：4-6个关键指标卡片，采用卡片式布局
- **实时图表区域**：折线图、柱状图展示趋势数据
- **快捷操作按钮**：常用功能快速入口
- **最新动态**：系统通知、用户活动时间线
- **系统状态**：服务器状态、在线用户数等

**背景色**：#F5F5F5（浅灰背景）

#### 2. 👥 用户管理页面
**布局结构**：
- 顶部搜索筛选栏
- 数据表格主体区域
- 底部分页组件

**主要元素**：
- **搜索栏**：用户名、手机号、注册时间筛选
- **操作工具栏**：批量操作、导出、新增用户按钮
- **用户列表表格**：头像、用户名、手机号、注册时间、状态、操作列
- **用户详情弹窗**：完整用户信息展示和编辑
- **状态标签**：活跃(绿色)、禁用(红色)、待审核(橙色)

**背景色**：#FFFFFF（白色背景）

#### 3. 📱 内容管理页面
**布局结构**：
- 左侧内容分类树
- 右侧内容列表和编辑区域

**主要元素**：
- **分类导航树**：内容分类层级结构
- **内容列表**：标题、类型、发布时间、状态、操作
- **富文本编辑器**：支持图片、视频、链接插入
- **媒体库**：图片、视频上传和管理
- **预览功能**：实时预览小程序端效果

**背景色**：#FAFAFA（极浅灰背景）

#### 4. 📊 数据分析页面
**布局结构**：
- 顶部时间筛选器
- 多维度图表展示区域
- 底部详细数据表格

**主要元素**：
- **时间选择器**：今日、昨日、本周、本月、自定义
- **核心指标卡片**：PV、UV、转化率、留存率
- **趋势图表**：折线图展示用户增长趋势
- **热力图**：用户行为热力图
- **排行榜**：热门页面、用户活跃度排行
- **数据导出**：Excel、PDF格式导出

**背景色**：#F8F9FA（浅蓝灰背景）

#### 5. ⚙️ 系统设置页面
**布局结构**：
- 左侧设置分类菜单
- 右侧设置表单区域

**主要元素**：
- **基础设置**：系统名称、Logo、联系方式
- **安全设置**：密码策略、登录验证、IP白名单
- **通知设置**：邮件通知、短信通知配置
- **备份设置**：自动备份时间、备份路径
- **主题设置**：界面主题、颜色方案选择

**背景色**：#FFFFFF（白色背景）

#### 6. 🔐 登录页面
**布局结构**：
- 居中登录表单
- 品牌Logo和背景图

**主要元素**：
- **登录表单**：用户名/邮箱、密码输入框
- **验证码**：图形验证码或短信验证码
- **记住密码**：复选框
- **登录按钮**：主要操作按钮
- **忘记密码**：链接跳转
- **品牌展示**：公司Logo、标语

**背景色**：渐变背景 (#667eea 到 #764ba2)

### 响应式设计
- **桌面端**：>= 1200px，完整布局展示
- **平板端**：768px - 1199px，侧边栏可折叠
- **移动端**：< 768px，抽屉式菜单，单列布局

### 交互规范
- **按钮状态**：默认、悬停、点击、禁用
- **加载状态**：骨架屏、加载动画
- **反馈提示**：成功、警告、错误消息提示
- **确认操作**：重要操作需二次确认
- **快捷键**：支持常用操作快捷键

### 无障碍设计
- **颜色对比度**：符合WCAG 2.1 AA标准
- **键盘导航**：支持Tab键导航
- **屏幕阅读器**：语义化HTML标签
- **字体大小**：最小14px，支持缩放
