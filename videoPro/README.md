# 智能图片/视频查重检测系统

## 项目概述
智能图片/视频查重检测系统是一个基于计算机视觉技术的内容重复检测工具，能够自动识别相似或重复的图片和视频文件。该系统采用前后端分离架构，提供高效、准确的媒体内容查重服务。

## 系统架构

### 前端
- **技术栈**：React + Vite + Ant Design
- **功能**：文件上传（支持拖拽、多选）、任务状态展示、结果可视化、参数配置
- **特点**：响应式设计，支持实时进度显示，直观的相似度评分展示

### 后端（双引擎支持）
1. **Python + OpenCV方案**
   - 基于感知哈希（pHash）和ORB特征匹配
   - 视频处理：提取关键帧并计算帧间相似度
   - 优势：成熟的计算机视觉库支持，识别准确率高

2. **Node.js + Video Duplicate Finder方案**
   - 基于FFmpeg的视频处理，实现高效帧提取
   - 优化的特征提取算法，对分辨率变化、水印等情况具有鲁棒性
   - 优势：Node.js生态集成，部署简单，性能优秀

## 技术原理

### 图片查重
- 采用pHash（感知哈希）生成64位特征哈希
- 计算汉明距离评估相似度
- 对复杂变换使用ORB特征点匹配增强识别能力

### 视频查重
- 使用OpenCV/FFmpeg提取视频关键帧
- 对关键帧序列进行相似度分析
- 综合计算帧集合间的平均相似度和最大相似度

### 相似度阈值
- 默认阈值：0.8（相似度>80%视为重复）
- 可在前端界面灵活配置

## 快速开始

### 前端部署

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 开发模式运行
npm run dev

# 生产环境构建
npm run build
```

### 后端部署（二选一）

#### Python后端

```bash
# 进入Python后端目录
cd backend/python

# 安装依赖
pip install -r requirements.txt

# 运行服务
python app.py
```

#### Node.js后端

```bash
# 进入Node.js后端目录
cd backend/node

# 安装依赖
npm install

# 运行服务
npm start
```

## 系统要求

- **前端**：现代浏览器，支持ES6+
- **Python后端**：Python 3.7+, OpenCV 4.5+
- **Node.js后端**：Node.js 14+, FFmpeg（需要安装在系统路径）

## 功能特性

- 🔍 支持图片与视频文件的相似度检测
- ⚙️ 可自定义相似度阈值和视频采样率
- 📊 提供详细的查重报告，包括相似度评分和重复组标识
- 🖥️ 直观的Web界面，支持批量文件上传和结果预览
- 🧠 双引擎设计，可根据需求选择不同的后端实现
- 🔄 支持图片与图片、视频与视频、图片与视频之间的跨类型比较

## API接口

### POST /detect
- **功能**：执行文件相似度检测
- **请求参数**：
  - `files`：多部分表单文件（支持多个文件）
  - `similarityThreshold`：相似度阈值（可选，默认0.8）
  - `frameSampleRate`：视频采样率（可选，默认10fps）
- **返回结果**：JSON格式的检测结果

### GET /health
- **功能**：服务健康检查
- **返回结果**：`{"status": "healthy"}`

## 注意事项

1. 系统默认支持的文件格式：
   - 图片：JPG、PNG、GIF、WebP等
   - 视频：MP4、AVI、MOV、MKV等

2. 文件大小限制：默认100MB，可在后端配置中调整

3. 视频处理可能较为耗时，请根据服务器性能调整并发数量

4. 对于大型视频文件，建议增加采样间隔以提高处理速度

## 性能优化

1. 使用多线程/异步处理提高并发能力
2. 实现结果缓存避免重复计算
3. 针对视频处理进行关键帧自适应采样
4. 支持分布式部署，扩展处理能力
