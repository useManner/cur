# 小程序后台管理系统 - React实现

## 🚀 项目概述

基于React + TypeScript + Ant Design实现的小程序后台管理系统，完全按照README.md中的设计规范进行开发。

## 📁 项目结构

```
src/
├── components/           # 公共组件
│   └── Layout/          # 布局组件
│       └── MainLayout.tsx
├── pages/               # 页面组件
│   ├── Login.tsx        # 登录页面
│   ├── Dashboard.tsx    # 仪表板首页
│   ├── UserManagement.tsx   # 用户管理
│   ├── ContentManagement.tsx # 内容管理
│   ├── DataAnalytics.tsx    # 数据分析
│   └── SystemSettings.tsx   # 系统设置
├── store/               # Redux状态管理
│   ├── store.ts         # Store配置
│   └── slices/          # Redux切片
│       ├── authSlice.ts     # 认证状态
│       ├── userSlice.ts     # 用户管理状态
│       └── systemSlice.ts   # 系统设置状态
├── types/               # TypeScript类型定义
│   └── index.ts
├── App.tsx              # 主应用组件
├── index.tsx            # 应用入口
└── index.css            # 全局样式
```

## 🎨 设计规范实现

### 配色方案
- **主色调**: #1890FF (深蓝色)
- **成功色**: #52C41A (绿色)
- **警告色**: #FA8C16 (橙色)
- **错误色**: #F5222D (红色)
- **背景色**: #F5F5F5 (浅灰色)
- **卡片背景**: #FFFFFF (白色)

### 页面布局
- **顶部导航栏**: 固定高度60px，深蓝色背景
- **左侧菜单栏**: 宽度240px，可折叠，白色背景
- **主内容区域**: 自适应宽度，白色背景，圆角6px

## 🛠 技术栈

### 前端框架
- **React 18.2.0** - 用户界面库
- **TypeScript 4.9.5** - 类型安全
- **Ant Design 5.11.0** - UI组件库

### 状态管理
- **Redux Toolkit 1.9.7** - 状态管理
- **React Redux 8.1.3** - React绑定

### 路由和导航
- **React Router DOM 6.18.0** - 客户端路由

### 图表可视化
- **ECharts 5.4.3** - 数据可视化
- **ECharts for React 3.0.2** - React集成

### 工具库
- **Axios 1.6.0** - HTTP客户端
- **Day.js 1.11.10** - 日期处理

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm start
```

### 3. 构建生产版本
```bash
npm run build
```
