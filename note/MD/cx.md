## 微信小程序项目文档：简账本

### 一、产品概述

- **产品名称**：简账本（可更改）
- **目标用户**：普通用户、学生、上班族、自由职业者
- **主要用途**：
  - 快速记录日常收入与支出
  - 可视化展示月度消费情况
  - 设置预算预警，管理财务习惯
  - 支持多账户（如微信钱包、银行卡、支付宝）

### 二、功能需求（PRD）

#### 1. 首页（账本总览）

- 今日收支金额
- 本月收支图表（饼图/柱状图）
- 快捷入口：记一笔、查看分类、导出账单

#### 2. 记账页面

- 支出 / 收入 切换
- 金额输入框
- 分类选择（吃饭、交通、购物等，可自定义）
- 支付方式（现金/微信/支付宝/银行卡）
- 时间选择（默认当前时间）
- 备注填写
- 支持语音输入（可选）

#### 3. 账单明细页面

- 展示每日流水列表
- 支持按月、分类、金额、关键词搜索
- 支持编辑、删除记录

#### 4. 分类管理

- 查看默认分类
- 添加/删除/修改自定义分类

#### 5. 设置页面

- 月度预算设置
- 多账户管理
- 数据导出（CSV）
- 数据备份（云端/本地）
- 夜间模式切换

#### 6. 用户账户功能（可选）

- 微信授权登录
- 数据云同步（绑定微信 OpenID）
- 多设备数据同步

### 三、技术实现建议

#### 技术栈推荐

- 框架：微信原生 / Taro / uni-app
- 状态管理：MobX / Pinia（视框架而定）
- 数据存储：`wx.setStorage`（本地） + 云开发数据库（可选）
- 图表组件：`ECharts for WeChat`

#### 页面结构建议

```
/pages
  ├── index/index            // 首页
  ├── record/record          // 记一笔
  ├── detail/detail          // 账单明细
  ├── category/category      // 分类管理
  └── settings/settings      // 设置页面
/components
  ├── chartView              // 图表组件
  └── recordForm             // 记账表单组件
/utils
  └── storage.ts             // 本地存储封装
/cloudfunctions
  └── backup/index.js        // 云备份函数（可选）
```

### 四、开发与上线流程

#### 1. 准备工作

- 注册小程序账号 https://mp.weixin.qq.com
- 创建小程序 AppID
- 安装微信开发者工具
- 创建项目并绑定 AppID
- 启用云开发（如使用）

#### 2. 开发阶段

- 页面开发
- 本地功能调试
- ECharts 图表调试
- 真机测试

#### 3. 提交体验版

- 上传体验版到微信
- 设置内测人员扫码体验

#### 4. 提交审核

- 填写小程序介绍和类目（"工具" > "记账"）
- 等待微信官方审核（1-3个工作日）

#### 5. 发布上线

- 审核通过后点击发布
- 发布后可扫码或微信搜索使用

### 五、未来可扩展功能

- 周/月消费报告自动推送
- 图表导出为图片分享
- 多人共享账本（如情侣/家庭账本）
- 小组件支持（添加到手机桌面）
